# round-robin
Round Robin competition site for UGGRA.

## Genral Flow

1. Invite competitor
2. fill brackets
3. Build matches
4. Enter results
5. GOTO 3 unitl finals
6. Show winner
7. log results for league

# Running
```$ docker-compose up -d```
